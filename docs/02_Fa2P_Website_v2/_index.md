## Tech-Stack

| Schicht | Technologie |
|---------|-------------|
| Sprache | PHP (prozedural, kein Framework) |
| Datenbank | MariaDB 10.6 (PDO, Prepared Statements) |
| Frontend | Bootstrap 4, jQuery 3.6, Popper.js |
| Bildergalerie | Lightbox.js |
| Template-Engine | Eigene Template-Klasse (`#variable#`-Syntax) |
| Analytics | eTracker, Google Analytics |
| Cookie-Consent | Cookiebot |
| Deployment | Apache auf Shared Hosting (kein Docker) |
