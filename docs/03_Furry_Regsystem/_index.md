# Furry Regsystem (RegSys)

Modulares, API-first Registrierungssystem für Furry-Events und Community-Veranstaltungen.
RegSys ersetzt die bestehende Pretix-Instanz mit voller Datenkontrolle, Offline-Check-in und Multi-Vereinsfähigkeit.
