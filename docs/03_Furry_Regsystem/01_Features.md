# Features

| Feature | Status | Beschreibung |
|---------|--------|--------------|
| Event-Management | Bereitgestellt | Events anlegen, veröffentlichen, archivieren; dynamische Zusatzfelder und Antwortvarianten pro Event |
| Online-Registrierung | Bereitgestellt | Öffentlicher Registrierungsflow mit konfigurierbaren Pflicht- und Optionalfeldern |
| Teilnehmenden-Verwaltung | Bereitgestellt | Listen, Status-Tracking (confirmed, waiting_list, checked_in, cancelled), manuelle Bearbeitung, CSV-Export |
| Offline-Check-in | Bereitgestellt | Vor-Ort-Check-in ohne Internetverbindung; delta-basierte Synchronisation mit Konfliktauflösung |
| Zahlungsintegration | Bereitgestellt | SumUp-Kartenzahlung online und über Solo-Terminal; webhook-verifizierte Statusaktualisierungen |
| Vor-Ort-Zusatzbuchungen | Bereitgestellt | Getränkeguthaben, Merch und andere Addon-Kataloge pro Event mit Zahlungsanbindung |
| Self-Service-Portal | Bereitgestellt | Teilnehmende verwalten eigene Daten, Registrierungen und fordern Datenexporte an |
| Multi-Vereinsfähigkeit | Bereitgestellt | Mandantentrennung über Organisations-Slug; eigenes Logo und Branding pro Organisation und Event |
| DSGVO-Compliance | Bereitgestellt | Einwilligungsverwaltung, Lösch- und Exportprozesse, Legal Holds, Audit-Log |
| Pretix-kompatible API | Bereitgestellt | Übergangs-Endpoints für bestehende Integrationen mit dem Pretix-Format |
| Benachrichtigungen | Bereitgestellt | Konfigurierbare E-Mail-Vorlagen mit Platzhaltern; gerätebasierte Zustellpräferenzen |
| Plattform-Administration | Geplant | Organisations- und Benutzerverwaltung, Rollen-Zuweisung, Einladungsflow |
